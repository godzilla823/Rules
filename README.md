## DivineEngine Core Special Edition

这是一个面向 Power Users、旨在给学习及定制规则的人一个**示例**的项目，也就是说这不再是一个面向大多数人的项目。

且需要注意的是，本项目中的内容极有可能会**随时**的修改变更，故而不建议直接使用。

### 支持



| 应用                                                         | 规则                                                         | 描述                                                         |
