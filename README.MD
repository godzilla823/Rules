## DivineEngine Core Special Edition

这是一个面向 Power Users 的**示例**项目，也就是说这不再是一个面向大多数人的项目。且项目中的文件及内容会根据测试实验而**随时**的修改变更，故需谨慎考虑直接使用本项目。
